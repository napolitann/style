# style

this library is a stylesheet for napolitann wiki

- language: SASS

## Getting Started

- if you do not have SASS, please visit a [SASS install page](http://sass-lang.com/install) and install it

  or

  ```
  npm install -g node-sass
  ```

- build
  ```
  // build all
  npm run build

  // build index
  npm run build:index

  // build clearfix
  npm run build:clearfix
  ```

- dev
  ```
  // alias 'npm run dev:index'
  npm run dev

  // watch index
  npm run dev:index

  // watch clearfix
  npm run dev:clearfix
  ```
